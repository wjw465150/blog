java生成pdf以及解决中文中文乱码问题

itext版本：5.4.0

下载地址：itext5.4：http://jaist.dl.sourceforge.net/project/itext/iText/iText5.4.0/itext-5.4.0.zip

                      extrajars：http://jaist.dl.sourceforge.net/project/itext/extrajars/extrajars-2.3.zip

其中extrajars中有支持中文的itext-asian.jar。

像工程中导入itextpdf-5.4.0以及itext-asian.jar即可支持中文。

示例代码如下：
原文参考自站长网：http://www.software8.co/wzjs/java/3319.html

Java代码:  
  1.           
  2. //1.新建document对象  
  3.         //第一个参数是页面大小。接下来的参数分别是左、右、上和下页边距。  
  4.         Document document = new Document(PageSize.A4, 50, 50, 50, 50);  
  5.         //2.建立一个书写器(Writer)与document对象关联，通过书写器(Writer)可以将文档写入到磁盘中。  
  6.         //创建 PdfWriter 对象 第一个参数是对文档对象的引用，第二个参数是文件的实际名称，在该名称中还会给出其输出路径。  
  7.         PdfWriter.getInstance(document, new FileOutputStream("F:\\ITextTest.pdf"));  
  8.         //3.打开文档  
  9.         document.open();          
  10.         //4.向文档中添加内容  
  11.         BaseFont bf = BaseFont.createFont("STSong-Light", "UniGB-UCS2-H", BaseFont.NOT_EMBEDDED);  
  12.         Font font = new Font(bf, 12, Font.NORMAL);  
  13.         document.add(new Paragraph("艹，生成了个pdf！！！",font));  
  14.         document.add(new Paragraph("我去我去，你妹啊！！！！！！！！！！！！！！！！！！！！！！！！！！！",   
  15.         FontFactory.getFont(FontFactory.COURIER, 14, Font.BOLD, new BaseColor(255, 150, 200))));  
  16.         //5.关闭文档  
  17.         document.close();
