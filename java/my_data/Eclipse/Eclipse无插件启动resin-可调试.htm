Eclipse无插件启动resin - 可调试

第一步 、配置resin 的xml文件

      <host id="" root-directory=".">
      <web-app id="/" document-directory="D:\myapp\pro" >
      </web-app>

      </host>

第二步、配置eclipse的debug

在eclipse中，选择菜单Run->debug...进入设置页面
New一个新的Java Application，然后设置相关的参数
1、ClassPath
   User Entries下添加resion.jar(RESIN_HOME\lib\下)
2、Main
   Main Class 设置为 com.caucho.server.resin.Resin

3、Argument
   1）program argument设置为 -conf D:\resin\conf\myconf.xml
   2）vm argument设置为：   
  -Dresin.home="D:\resin"
  -Djava.util.logging.manager=com.caucho.log.LogManagerImpl
