jatoolsPrinter 免费版与收费版区别是什么？

总体来说，区别表现在三个方面：
一、 以下是free版不支持的功能(收费版支持)：
直接打印url;
同时进行多个url打印;
自动创建自定义纸张;
表格的自动分页；
条形码的显示、打印
支持除ie外的浏览器，如firefox,chrome,safar等。
... 更多功能区别，参照这里。
二、 收费版的服务形式有：qq远程支持，电话，Email，bbs等，free版只通过bbs进行技术支持。
三、 收费版除买断版外，均按服务器的ip或域名购买，而免费版是免费的，不限ip或域名。
