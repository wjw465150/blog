使用Spring中 Commons FileUpload 实现文件上传 在 request 到 MultipartHttpServletRequest转型的时候会出现 转型异常 ClassCastException.

MultipartHttpServletRequest multipartRequest = (MultipartHttpServletRequest) request

 

有效的解决办法之一：

Java代码  
  1. CommonsMultipartResolver commonsMultipartResolver = new CommonsMultipartResolver(request  
  2. .getSession().getServletContext());  
  3. commonsMultipartResolver.setDefaultEncoding("utf-8");  
  4. MultipartHttpServletRequest multipartRequest = commonsMultipartResolver.resolveMultipart(request);  

 可实现 request 到 MultipartHttpServletRequest 的转型。
