xmanager链接linux的方法
介绍xmanager链接linux的方法。

服务器：Red Hat Linux

终端：Xmanager 1.3.9

使用方法： 

服务器上操作：

－－－－－－－－－－－－－－－－－－－－－－－

1. 修改 /etc/X11/xdm/Xaccess，将下面的行： 

#* # any host can get a login window 

改为： 

* # any host can get a login window 

－－－－－－－－－－－－－－－－－－－－－－－－

2. 修改/etc/X11/gdm/gdm.conf，找到下面的信息： 

[xdmcp] 

Enable=0 或Enable=false 

修改为： 

[xdmcp] 

Enable=1 或Enable=true 

并确保以下信息存在： 

Port=177 

－－－－－－－－－－－－－－－－－－－－－－－－

3: 修改/etc/inittab，将 

id:3:initdefault: 

修改为： 

id:5:initdefault: 

并将最后一行改为： 

x:5:respawn:/usr/bin/gdm 

－－－－－－－－－－－－－－－－－－－－－－－－－

4: 确保/etc/X11/xdm/Xservers的属性为444，/etc/X11/xdm/Xsetup_0的属性为755。

注意:如果还连不上,看一下:
/etc/X11/xdm/xdm-config 文件，注释掉这一样。   
Display Manager .Requestport 0
