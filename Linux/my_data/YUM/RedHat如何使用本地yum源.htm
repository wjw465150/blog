redhat如何使用本地yum源
在/etc/yum.repos.d目录里建立一个文件，文件名任意，扩展名一定是repo，文件内容如下：
[base]
name=base RPM Repository for RHEL5
baseurl=file:///media/CDROM/Server
enabled=1
gpgcheck=0

修改server上的/usr/lib/python2.4/site-packages/yum/yumRepo.py文件，这个文件的411行原来的内容是：

remote = url + '/' + relative
修改为：
remote = "file:///media/CDROM/Server" + '/' + relative
然后运行命令：
＃yum clean all 
#yum update
就ok了。
