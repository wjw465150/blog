cd repository/hooks，找到pre-commit.tmpl文件，去掉后缀.tmpl， 编辑pre-commit文件： 
windows: 重命名为pre-commit.bat 

Java代码  
  1. @echo off  
  2. setlocal  
  3. set REPOS=%1  
  4. set TXN=%2  
  5. rem check that logmessage contains at least 10 characters  
  6. rem .....代表5个字符  
  7. svnlook log "%REPOS%" -t "%TXN%" | findstr ".........." > nul  
  8. if %errorlevel% gtr 0 goto err  
  9. exit 0  
  10. :err  
  11. echo Empty log message not allowed. Commit aborted! 1>&2  
  12. exit 1  


linux 
Java代码  
  1. #!/bin/sh  
  2. REPOS="$1"  
  3. TXN="$2"  
  4. SVNLOOK=/usr/bin/svnlook  
  5. # check that logmessage contains at least 10 alphanumeric characters  
  6. LOGMSG=`$SVNLOOK log -t "$TXN" "$REPOS" | grep "[a-zA-Z0-9]|[^\x00-\xff]" | wc -c`  
  7. if [ "$LOGMSG" -lt 10 ];  
  8. then  
  9.    echo -e "comment count:$LOGMSG \nEmpty log message not allowed. Commit aborted!" 1>&2  
  10.   exit 1  
  11. fi
