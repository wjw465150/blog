在Hadoop2.0中MapReduce程序的都需要继承org.apache.hadoop.mapreduce.Mapper 和 org.apache.hadoop.mapreduce.Reducer这两个基础类，来定制自己的mapreduce功能,源码中主要的函数如下

  1.   public void run(Context context) throws IOException, InterruptedException {  
  2.     setup(context);     // Called once at the beginning of the task.  
  3.     while (context.nextKey()) {  
  4.       reduce(context.getCurrentKey(), context.getValues(), context);  
  5.     }  
  6.     cleanup(context);   // Called once at the end of the task.  
  7.   }  
  8.   
  9.   /** 
  10.    * This method is called once for each key. Most applications will define 
  11.    * their reduce class by overriding this method. The default implementation 
  12.    * is an identity function. 
  13.    */  
  14.   protected void reduce(KEYIN key, Iterable<VALUEIN> values, Context context  
  15.                         ) throws IOException, InterruptedException {  
  16.     for(VALUEIN value: values) {  
  17.       context.write((KEYOUT) key, (VALUEOUT) value);  
  18.     }  
  19.   }  

       在Mapper和Reducer类中，都有一个run()方法不断提供(key,value)来调用map()和reduce()函数来处理，我们一般只需重写其中的map和reduce方法。在mapreduce中只有支持序列化的类才能作为键值，其中的key还必须要是可比较的，故 key要实现WritableComparable接口，value只需要实现Writable接口。

  18.     public static class WordCountMapper extends Mapper<Object,Text,Text,IntWritable> {  
  19.           
  20.         private static final IntWritable one =  new IntWritable(1);  
  21.           
  22.         private Text word = new Text();  
  23.   
  24.         protected void map(Object key, Text value, Context context)  
  25.                 throws IOException, InterruptedException {  
  26.             String line = value.toString();  
  27.             StringTokenizer words = new StringTokenizer(line);  
  28.             while(words.hasMoreTokens()) {  
  29.                 word.set(words.nextToken());  
  30.                 context.write(word, one);  
  31.             }  
  32.         }  
  33.     }  
  34.       
  35.     public static class WordCountReducer extends Reducer<Text,IntWritable,Text,IntWritable> {  
  36.   
  37.         private IntWritable totalNum = new IntWritable();  
  38.           
  39.         @Override  
  40.         protected void reduce(Text key, Iterable<IntWritable> values,Context context)  
  41.                 throws IOException, InterruptedException {  
  42.            int sum = 0;  
  43.            Iterator<IntWritable> it = values.iterator();  
  44.                    while(it.hasNext()) {  
  45.                    sum += it.next().get();  
  46.                    }  
  47.                    totalNum.set(sum);  
  48.                    context.write(key,totalNum);   
  49.         }  
  50.     }
