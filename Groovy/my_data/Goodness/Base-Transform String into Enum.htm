Groovy Goodness: Transform String into Enum

After reading Groovy, State of the Union - Groovy Grails eXchange 2010 by Guillaume Laforge I discovered that in Groovy 1.7.6 we can transform a String into a Enum value. We can use type coersion or the as keyword to turn a String or GString into a corresponding Enum value (if possible).


00.enum Compass {
01.NORTH, EAST, SOUTH, WEST
02.}
03. 
04.// Coersion with as keyword.
05.def north = 'NORTH' as Compass
06.assert north == Compass.NORTH
07. 
08.// Coersion by type.
09.Compass south = 'south'.toUpperCase()
10.assert south == Compass.SOUTH
11. 
12.def result = ['EA', 'WE'].collect {
13.// Coersion of GString to Enum.
14."${it}ST" as Compass
15.}
16.assert result[0] == Compass.EAST
17.assert result[1] == Compass.WEST
