假设我们有两个List，分别是a和b，a和b的长度相同，现在我们想把a的每个元素作为key，b中对应的元素作为value，构造一个Map，应该怎么写呢？ 

最直接的想法，先建立一个空的Map，然后往里面添加key-value对

代码片段(2)[全屏查看所有代码]

1. [代码]标准的     

1 def a = [1, 2, 3]  
2 def b = [4, 5, 6]  
3    
4 def m = [:]  
5 [a, b].transpose().each { k, v -> m += [(k): v] }  
6 assert m == [1: 4, 2: 5, 3: 6]

2. [代码]通过inject方法，把代码缩减为一行     

1 def m = [a, b].transpose().inject([:]) { s, l -> s + [(l[0]): l[1]] }  
2 assert m == [1: 4, 2: 5, 3: 6]
